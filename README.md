# Deep BSDE Portfolio Optimisation

**MSc Financial Mathematics Dissertation (Highest Mark in Cohort)**

## Overview

This repository contains my MSc dissertation and Python implementation of a Deep BSDE framework for time-consistent mean-variance portfolio optimisation under transaction costs.

The project combines stochastic control theory, backward stochastic differential equations (BSDEs), deep learning and historical market data to optimise dynamic portfolio allocation.

## Research Objective

Develop a scalable Deep BSDE framework capable of solving continuous-time mean-variance portfolio optimisation with realistic transaction costs.

## Methodology

- Deep 2BSDE implementation (PyTorch)
- Basak & Chabakauri benchmark
- Li & Ng discrete benchmark
- Historical backtesting
- Bootstrap robustness analysis
